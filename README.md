# Java Practice Projects

This repository contains Java practice projects focusing on backend development.  
The projects follow my daily learning goals from operators, OOP, collections to Spring Boot basics.

## Folder structure

- **Day-01_Operators_UserInput**: Java basics with operators and user input.
- **Day-02_Conditionals_Loops**: Conditionals and loops applied to insurance scenarios.
- **Day-03_Arrays_Methods_Variables**: Arrays, methods, and variables practice.
- **Day-04_BasicOOP**: Basic Object-Oriented Programming concepts with insurance policy modeling.
- **Day-05_AdvancedOOP**: Advanced OOP concepts including encapsulation and interfaces.
- **Day-06_ArrayLists_ExceptionHandling**: ArrayLists and exception handling in management.
